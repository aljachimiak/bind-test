What is React.JS?

React is a JavaScript library for creating user interfaces by Facebook and Instagram. Many people choose to think of React as the V in MVC.



We [facebook engineering team] built React to solve one problem: building large applications with data that changes over time.

What does React do?

React uses a concept called the Virtual DOM that selectively renders subtrees of nodes based upon state changes.



It does the least amount of DOM manipulation possible in order to keep your view up to date.

How?

Imagine you had an object that you modeled around a real person.

The object has every relevant property a person could possibly have, and mirrored the persons current state. This is basically what React does with the DOM when it creates a Virtual DOM.

Now think about if you took that object and made some changes to the state of the object. Added a mustache, some sweet biceps and Steve Buscemi eyes.

Regular method React
  • Takes the real person and rebuilds them from the ground up.
  1. React runs a “diffing” algorithm, which identifies what has changed.
  2. Reconciliation, where it updates the DOM with the results of diff.
  3. Only change would be the face and the arms.
